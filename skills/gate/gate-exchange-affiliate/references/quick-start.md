# Quick Start Guide

Get started with the Gate Exchange Affiliate skill in 5 minutes.

## Prerequisites

1. **Partner Account**: You must have partner/affiliate privileges on Gate Exchange
2. **API Credentials**: Valid API key and secret with partner access
3. **MCP Server**: Gate MCP server installed and configured

## Installation

### Step 1: Install Gate MCP Server

```bash
npm install -g gate-mcp
```

### Step 2: Configure MCP

Create or update your MCP configuration:

```json
{
  "mcpServers": {
    "gate": {
      "command": "npx",
      "args": ["-y", "gate-mcp"],
      "env": {
        "GATE_BASE_URL": "https://api.gateio.ws",
        "GATE_API_KEY": "your-api-key",
        "GATE_API_SECRET": "your-api-secret"
      }
    }
  }
}
```

### Step 3: Load the Skill

Add the skill to your AI assistant by referencing the skill directory.

## Important Notes

### Time Limitations

- **API Limit**: Each request can query maximum 30 days
- **Skill Support**: Up to 180 days (automatically splits into multiple requests)
- **Default**: Last 7 days when no time specified

### Automatic Request Splitting

When you request data for more than 30 days, the skill automatically:
1. Splits the time range into 30-day segments
2. Makes multiple API calls
3. Merges and aggregates the results
4. Presents unified data

Example:
- Request: "Last 60 days commission"
- Behind the scenes: 2 API calls (30 days each)
- Result: Combined 60-day data

### Data Metrics

The skill provides 5 core metrics:

1. **Commission Amount**: Your rebate earnings
2. **Trading Volume**: Total trading by referred users
3. **Net Fees**: Fees generated by your referrals
4. **Customer Count**: Total number of subordinates
5. **Trading Users**: Number of actively trading users

## Troubleshooting

### "Not an affiliate" Error

**Problem**: You receive an error about not having affiliate privileges.

**Solution**: 
1. Verify your account has partner role
2. Apply for affiliate program at: https://www.gate.com/referral/affiliate
3. Check API credentials have correct permissions

### No Data Returned

**Problem**: Queries return 0 or no data.

**Possible Causes**:
- No referred users have traded in the specified period
- Time range is in the future
- Account recently approved (data not yet available)

### Time Range Errors

**Problem**: "Time range exceeds limit" error.

**Solution**:
- Maximum single query: 30 days (handled automatically)
- Maximum total range: 180 days
- For older data, contact support

## Best Practices

1. **Start with Overview**: Use "show my affiliate data" to get familiar with your metrics

2. **Use Specific Dates**: Instead of "last month", use "March 1 to March 31" for precision

3. **Monitor Regularly**: Check daily or weekly for consistent tracking

4. **Analyze Trends**: Compare periods to identify growth patterns

5. **Focus on Active Users**: Track "trading users" not just total customers
