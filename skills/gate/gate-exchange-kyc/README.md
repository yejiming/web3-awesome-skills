# Gate Exchange KYC Portal Skill

## Overview

Guides users to the Gate KYC portal to complete identity verification. Provides the portal link and brief instructions only; does not perform verification.

### Core Capabilities

| Capability | Description |
|------------|-------------|
| KYC portal entry | Provides the official KYC portal URL and tells user to log in, open the link, and complete verification on the portal. |
