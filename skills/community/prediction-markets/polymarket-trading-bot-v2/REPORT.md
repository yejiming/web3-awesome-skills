## 🎯 核心功能

### 1. 市场监控（auto_bot_v2.py）
- **监控市场**：BTC/ETH 5分钟预测市场
- **扫描频率**：每 2 秒轮询一次
- **预热机制**：市场出现即预加载 PTB（Price to Beat）
- **触发时机**：市场结束前 40 秒自动触发分析
- **自动记录**：所有决策保存到 `logs/decisions_v2.jsonl`

### 2. PTB 获取（playwright_ptb.py）
- **技术方案**：Playwright 常驻浏览器
- **性能**：首次 ~12 秒，复用浏览器 ~9 秒
- **4 层 Fallback**：
  1. Polymarket API
  2. HTML 静态抓取
  3. Playwright 动态渲染
  4. Binance 实时价格近似
- **优化**：资源拦截（屏蔽图片/字体/分析脚本）

### 3. 技术指标计算（indicators.py）
- **纯 Python 实现**（无需 talib）
- **支持指标**：
  - EMA(9/21) - 指数移动平均
  - RSI - 相对强弱指数
  - MACD - 移动平均收敛发散
  - 布林带 - 波动率通道
  - ATR - 平均真实波幅
  - 成交量分析

### 4. AI 评分模型（ai_model_v2.py）
**权重分配**：
- 位置信号（50%）：当前价 vs PTB，用 ATR 归一化
- 短期动量（30%）：近 3 分钟趋势 + 1 分钟微观
- RSI 极端（10%）：超买超卖修正
- 成交量（10%）：放量确认

**核心逻辑**：
- 位置 + 动量方向一致 → 强信号
- 位置 + 动量矛盾 → 大幅降分

**输出**：
- 方向：UP / DOWN
- 置信度：0-100%
- 期望值（EV）

### 5. 决策引擎（ai_analyze_v2.py）
**下注条件**：
- 置信度 ≥ 55%
- 期望值 EV > 0.05
- 赔率 < 0.90

**决策逻辑**：
- 基于期望值（EV）而非固定阈值
- 自动记录所有决策到日志

### 6. 回测验证（backtest_accuracy.py）
- 自动获取市场最终结果
- 按置信度/币种/方向分组统计
- 生成详细准确率报告
- 保存完整结果到 JSON
