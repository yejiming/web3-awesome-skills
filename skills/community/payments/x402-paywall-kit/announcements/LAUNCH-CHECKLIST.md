- [ ] NPM packages published (`npm publish` for all 3 packages)
- [ ] ClawHub full skill published (`cd x402-agent && clawhub publish`)
- [ ] ClawHub free skill published (`cd x402-agent-free && clawhub publish`)
- [ ] ClawMart listing live (follow `clawmart/PUBLISHING.md`)
- [ ] taraquinn.ai product page live (follow `taraquinn-integration/INTEGRATION.md`)
