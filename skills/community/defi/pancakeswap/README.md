# PancakeSwap

> High APR is not the same thing as high-quality liquidity.

PancakeSwap is a protocol-truth-enforcer skill for evaluating pools and LP structures before capital is deployed.

## What it does

This skill helps you:
- assess pool depth and fragility
- evaluate concentrated-liquidity range risk
- reason about fee-tier suitability
- screen pool quality before LP deployment
- decide whether to deploy, widen, reduce, monitor, or avoid

## What it does NOT do

- execute trades
- connect to wallets
- guarantee safety
- replace smart contract audits
