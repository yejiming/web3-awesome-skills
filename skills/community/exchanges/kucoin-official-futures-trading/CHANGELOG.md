# Changelog

All notable changes to the KuCoin Futures Trading skill will be documented in this file.

## [1.0.0] - 2026-03-14

### Added

- Initial release of the KuCoin Futures Trading skill.
- Pro API support for placing, cancelling, and querying futures orders.
- Pro API support for managing positions (UTA).
- Pro API support for futures market data (public endpoints).
- Classic API support for futures market data.
- Classic API support for futures orders including TP/SL orders.
- Classic API support for position management (margin mode, position mode, leverage).
- Classic API support for risk limit management.
- Funding rate and funding fee history endpoints.
- Disconnection Protection (DCP) endpoints.
- HMAC-SHA256 authentication support.
- Security rules for credential handling and trade confirmation.
