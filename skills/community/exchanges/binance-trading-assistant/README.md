# Binance Trading Assistant

**Monitor your Binance account, track positions, and get real-time alerts — all through your AI assistant.**

## Features

- 💰 **Balance Monitoring** - Check spot & futures balances instantly
- 📊 **Position Tracking** - Monitor open positions with P&L
- 🔔 **Price Alerts** - Get notified when prices hit your targets
- 📈 **Market Summary** - Quick overview of your portfolio

## Installation

```bash
clawhub install binance-trading-assistant
```

## Configuration

Set your Binance API credentials:

```bash
export BINANCE_API_KEY="your_api_key"
export BINANCE_API_SECRET="your_api_secret"
```

## Usage

Ask your assistant:
- "Check my Binance balance"
- "Show my open positions"
- "What's my total P&L?"
- "Alert me when BTC hits $100k"
