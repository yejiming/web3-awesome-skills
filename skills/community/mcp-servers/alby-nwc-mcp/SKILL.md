# Alby NWC MCP Server

Official MCP server from Alby (Lightning wallet).

Official Alby Nostr Wallet Connect MCP server for Lightning Network payments, invoices, and balance management. Enables AI agents to interact with Lightning Network wallets through the Nostr Wallet Connect protocol for sending and receiving payments.

## Installation

```bash
# From the official repo:
git clone https://github.com/getalby/nwc-mcp-server
cd nwc-mcp-server
npm install
```
