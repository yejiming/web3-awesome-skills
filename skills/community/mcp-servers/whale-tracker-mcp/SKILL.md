# Whale Tracker MCP Server

Community MCP server from kukapay.

Crypto whale tracker MCP for monitoring large wallet movements and whale transaction alerts. Track significant on-chain movements, identify whale wallets, and receive alerts on large transactions across blockchain networks.

## Installation

```bash
# From the official repo:
git clone https://github.com/kukapay/whale-tracker-mcp
cd whale-tracker-mcp
npm install
```
