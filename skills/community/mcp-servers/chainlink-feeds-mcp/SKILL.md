# Chainlink Feeds MCP Server

Community MCP server from kukapay.

Chainlink price feeds MCP server for real-time access to decentralized on-chain oracle price data. Query Chainlink's decentralized oracle network for reliable, tamper-proof price feeds used across DeFi protocols.

## Installation

```bash
# From the official repo:
git clone https://github.com/kukapay/chainlink-feeds-mcp
cd chainlink-feeds-mcp
npm install
```
