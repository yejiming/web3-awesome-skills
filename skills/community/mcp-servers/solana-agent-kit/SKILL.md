# Solana Agent Kit

Official agent toolkit from SendAI.

An open-source toolkit for connecting AI agents to Solana protocols. Any agent, using any model can autonomously perform 60+ Solana actions: trade tokens, launch new tokens, lend assets, send compressed airdrops, execute blinks, launch tokens on AMMs, bridge tokens across chains, and more. Integrates with Jupiter, Raydium, Orca, Meteora, Pump.fun, Jito, and many other Solana protocols.

## Installation

```bash
# From the official repo:
git clone https://github.com/sendaifun/solana-agent-kit
cd solana-agent-kit
npm install
```

Or install via npm:

```bash
npm install solana-agent-kit
```
