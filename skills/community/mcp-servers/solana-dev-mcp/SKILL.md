# Solana Dev MCP Server

Official MCP server from the Solana Foundation.

Official Solana Foundation developer MCP demo for Solana ecosystem documentation and developer tooling. Separate from solana-mcp-official, this server focuses on developer experience and provides access to Solana development resources and documentation.

## Installation

```bash
# From the official repo:
git clone https://github.com/solana-foundation/solana-dev-mcp
cd solana-dev-mcp
npm install
```
