# Lightning MCP Server

Community MCP server from AbdelStark.

Lightning Network MCP server for LN node management, invoices, channels, and payment routing. Interact with Lightning Network nodes to manage channels, create and pay invoices, and analyze payment routing across the network.

## Installation

```bash
# From the official repo:
git clone https://github.com/AbdelStark/lightning-mcp
cd lightning-mcp
npm install
```
