# GOAT (Great Onchain Agent Toolkit)

Official toolkit from GOAT SDK.

GOAT is the largest agentic finance toolkit for AI agents. Create agents that can send and receive payments, purchase physical and digital goods and services, earn yield, bet on prediction markets, purchase crypto assets, tokenize assets, and get financial insights. Supports 200+ tool integrations, multiple chains and wallets, and works with any agent framework. Available in both TypeScript and Python.

## Installation

```bash
# From the official repo:
git clone https://github.com/goat-sdk/goat
cd goat
npm install
```

Or install via npm:

```bash
npm install @goat-sdk/core
```
