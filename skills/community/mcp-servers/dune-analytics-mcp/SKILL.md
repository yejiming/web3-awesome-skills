# Dune Analytics MCP Server

Community MCP server from kukapay.

Dune Analytics MCP server for executing queries and retrieving on-chain analytics dashboards. Run SQL queries against blockchain data, access pre-built dashboards, and retrieve comprehensive on-chain analytics through Dune's data platform.

## Installation

```bash
# From the official repo:
git clone https://github.com/kukapay/dune-analytics-mcp
cd dune-analytics-mcp
npm install
```
