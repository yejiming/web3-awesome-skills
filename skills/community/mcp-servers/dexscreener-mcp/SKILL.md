# DexScreener MCP Server

Community MCP server from openSVM.

DexScreener MCP server for real-time DEX pair data, charts, and token analytics across all chains. Access DexScreener's comprehensive DEX analytics including pair information, price charts, and cross-chain token data.

## Installation

```bash
# From the official repo:
git clone https://github.com/opensvm/dexscreener-mcp-server
cd dexscreener-mcp-server
npm install
```
