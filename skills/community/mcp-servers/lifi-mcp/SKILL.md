# LI.FI MCP Server

Official MCP server from LI.FI.

MCP server for the LI.FI protocol, enabling AI agents to perform cross-chain token swaps and bridging across multiple blockchain networks. LI.FI aggregates bridges and DEXs to find optimal routes for cross-chain transactions.

## Installation

```bash
# From the official repo:
git clone https://github.com/lifinance/lifi-mcp
cd lifi-mcp
npm install
```
